Privacy Policy for Alex WHOOP Data

Alex WHOOP Data is a personal application created solely to access and analyze my own WHOOP data.

Information Collected

This application may access the following information from my WHOOP account:

Recovery score
Heart rate variability (HRV)
Resting heart rate
Sleep metrics
Workout information
Daily strain
Body measurements (if available)
How Information Is Used

The information is used exclusively to:

View my personal health and fitness data
Generate personal training and recovery insights
Track long-term trends
Integrate with personal tools such as ChatGPT for coaching and analysis
Data Sharing

No data is sold, rented, shared, or disclosed to third parties.

The application is intended for personal use only.

Data Storage

Any retrieved data is stored only as necessary to provide personal analysis.

Reasonable security measures are used to protect stored data.

Third-Party Services

This application accesses data provided by WHOOP through the official WHOOP API.

WHOOP is responsible for collecting and maintaining your WHOOP account data according to its own Privacy Policy.

Your Control
